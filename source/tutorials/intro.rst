简介
======================================================  

“开元编程开发套件”是可用于Python教学、创客项目制作的可编程电子套件，
其由主控和功能模块组成。


.. figure:: 主控2.png 
   :width: 400
   :align: center

主控:
	+ 开元

功能模块：
    + 1. 按键组
    + 2. 电位器
    + 3. 温湿度传感器 
    + 4. 超声波测距传感器
    + 5. 可编程全彩LED
    + 6. 电机风扇 
    + 7. 语音识别模块
