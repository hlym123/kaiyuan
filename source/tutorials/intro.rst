1. 简介
++++++++++++++++++++++++++++++++++++++++++++++++++++++ 
“开元 MicroPython 开发套件”是可用于Python教学、创客项目制作的可编程电子套件，
其由控制器和功能模块组成。


控制器:
	+ 1. MicroPython Pico 主控 

功能模块：
    + 1. 按键组
    + 2. 电位器
    + 3. 温湿度传感器 
    + 4. 超声波测距传感器
    + 5. 可编程全彩LED
    + 6. 电机风扇 
    + 7. 语音识别模块
